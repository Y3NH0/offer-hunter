# Base Agent

*2024 - Present | ITRI 專案*

- **Tech Stack:** LangChain, LangGraph, PGVector, LiteLLM, Arize Phoenix, FastAPI

## Contributions (zh)

- 設計並開發 Agent 專案的核心模組，提供可重用且可擴展的元件，讓開發者能專注於 domain logic，而非底層 Agent 架構實作，統一並加速團隊開發 Agent 專案
- 基於 LangChain 與 LangGraph 建立模組化抽象層，封裝 context engineering、RAG、memory、tool 整合與 HITL 等核心模組
- 設計並實作 Dynamic Node Config 系統，支援在 runtime 動態注入 Prompt、模型與參數，並可客製化不同專案需求
- 建立基礎架構與應用邏輯分離的設計，支援 Agent 系統的可擴展性與長期維護
- 開發 SKILL for claude code，讓同仁可以更快基於此基礎建設進行開發

## Contributions (en)

- Design and develop the core modules of the Agent platform, providing reusable and extensible components that allow developers to focus on domain logic instead of low-level Agent infrastructure, thereby standardizing and accelerating team development of Agent-based projects.
- Build a modular abstraction layer based on LangChain and LangGraph, encapsulating key capabilities such as context engineering, RAG, memory, tool integration, and human-in-the-loop (HITL).
- Design and implement a Dynamic Node Configuration system that supports runtime injection of prompts, models, and parameters, enabling flexible customization across different project requirements.
- Establish a clear separation between infrastructure and application logic to ensure scalability and long-term maintainability of Agent systems.
- Develop SKILLs for Claude Code to enable team members to rapidly build on top of this foundational framework.
