# Cloth Tagger

*2025-10 - Present | Side Project, work w/ my friend*

- **Tech Stack:** LangChain, LangGraph, VLM, YOLO

## Contributions (zh)

- 建立 VLM 標籤系統，自動從服裝圖片生成結構化屬性
- 基於 LangChain 與 LangGraph 設計可重用的 Agent 框架，支援模組化開發
- 建立混合式 AI pipeline，透過 LLM 生成標籤擴充資料集，並訓練 CV 模型（如 YOLO）支援高效推論

## Contributions (en)

- Built a vision-language tagging system to generate structured fashion attributes from images
- Designed a reusable agent framework based on LangChain and LangGraph for modular LLM application development
- Established a hybrid AI pipeline using LLM-generated labels to bootstrap datasets, followed by training CV models (e.g., YOLO) for scalable inference
