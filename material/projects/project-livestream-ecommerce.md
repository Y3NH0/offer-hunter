# Livestream E-Commerce AI Customer Service Agent

*Sep 2024 - Present | ITRI Project*

> NOTE: 這個專案算是我覺得比較沒價值的，只是 material 這邊先保留，如果真的有一些電商相關的公司機會的話，履歷我就會放這個，但放之前我也要再記得重新調整一下說辭包裝

- **Overview:** Designed core AI Workflow delivering automated customer service and real-time comment analysis for livestream e-commerce
- **Tech Stack:** LangChain, LangGraph, LLM

## Key Features

- **DM Analysis Agent:** Multi-step LLM workflow handling multi-intent interleaved conversations, automatically extracting intents and crawling order/inventory information
- **Livestream Comment Analysis Agent:** Real-time LLM classification and summarization with dynamic category configuration
- **Reasoning Output:** Enhanced intent identification accuracy

## Impact

Successfully delivered production use cases, enabling partner companies to automate customer service and allowing streamers to monitor chat highlights in real time
