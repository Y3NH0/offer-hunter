# 🇹🇼 台灣 AI Engineer 職缺市場報告：AI Agent & LLM 相關職位（2025-2026）

> 報告產出日期：2026-03-20

---

## 1. 市場概況

- 根據 104 人力銀行數據，AI 相關職缺在 2025 年 11 月已達 **99,000 筆**，年增 **38%**（前年 71,000 筆）
- IT 服務業每年新增約 **4,500 個 AI 人才職位**
- 廣達、鴻海、英業達、華碩、和碩、緯創、仁寶等大廠 2026 年合計預計招聘 **4,000+ 工程師**，多數聚焦 AI
- 市場存在顯著的 **人才缺口**，企業積極海外招募並提供薪資溢價

## 3. 技術能力需求

### 3.1 程式語言

| 語言 | 重要性 | 說明 |
|---|---|---|
| **Python** | ⭐⭐⭐⭐⭐ 必備 | 幾乎所有職缺都要求 |
| **SQL** | ⭐⭐⭐⭐ 常見 | 資料管道與分析 |
| **JavaScript / TypeScript** | ⭐⭐⭐ 加分 | 全端 AI 應用、MCP Server 開發 |
| **C/C++** | ⭐⭐⭐ 視公司 | 硬體相關公司（台積電、聯發科、趨勢科技） |
| **Go / Java** | ⭐⭐ 偶爾 | 高效能後端系統 |

### 3.2 LLM & AI 框架

| 框架 / 工具 | 重要性 | 說明 |
|---|---|---|
| **LangChain / LangGraph** | ⭐⭐⭐⭐⭐ | 最常被提及的 AI Agent 框架，多數職缺要求「精通」 |
| **LlamaIndex** | ⭐⭐⭐⭐ | 常與 LangChain 並列，用於 RAG pipeline |
| **Hugging Face Transformers** | ⭐⭐⭐⭐ | 模型微調與部署必備 |
| **PyTorch** | ⭐⭐⭐⭐ | 主流深度學習框架（勝過 TensorFlow） |
| **OpenAI API / Anthropic API** | ⭐⭐⭐⭐ | LLM API 整合 |
| **CrewAI** | ⭐⭐⭐ | 新興多 Agent 框架，2026 年職缺中越來越常見 |
| **scikit-learn** | ⭐⭐⭐ | 傳統 ML 任務 |

### 3.3 LLM 專業技能

| 技能 | 重要性 | 說明 |
|---|---|---|
| **RAG（檢索增強生成）** | ⭐⭐⭐⭐⭐ | 最常見的生產環境模式：chunking 策略、embedding 模型、retrieval 方法、reranking |
| **Prompt Engineering** | ⭐⭐⭐⭐⭐ | 系統化的 prompt 設計與 LLM 輸出優化 |
| **AI Agent 架構** | ⭐⭐⭐⭐⭐ | reasoning、planning、tool use、memory management 模式 |
| **Function Calling / Tool Calling** | ⭐⭐⭐⭐ | 讓 LLM 與外部 API 互動 |
| **Fine-Tuning** | ⭐⭐⭐⭐ | LoRA、QLoRA、PEFT 等微調方法 |
| **MCP（Model Context Protocol）** | ⭐⭐⭐⭐ | Anthropic 的 AI 工具整合開放標準，2026 年快速崛起 |
| **Transformer 架構** | ⭐⭐⭐⭐ | 深入理解 GPT、BERT、T5、LLaMA 模型家族 |
| **模型評估** | ⭐⭐⭐ | TruLens、RAGAS、DeepEval 等 RAG 品質評估框架 |
| **RLHF** | ⭐⭐⭐ | 人類反饋強化學習，用於模型對齊 |
| **AI 安全 & Red Teaming** | ⭐⭐⭐ | 內容過濾、偏差偵測、負責任 AI |
| **多模態 AI** | ⭐⭐⭐ | 視覺-語言模型、跨模態應用 |

## 4. 軟實力 & 領域知識

### 軟實力
- **溝通能力** — 能向非技術人員和高層說明複雜 ML 概念
- **跨部門協作** — 帶領多團隊專案
- **自學能力** — 快速適應 AI 領域的快速變化
- **英文能力** — 閱讀論文是基本要求，外商需流利口說
- **專案管理** — 特別是資深/主管職位
- **知識分享文化** — 多個職缺特別提到「熱衷分享新知」

### 高薪領域知識

| 領域 | 說明 |
|---|---|
| **金融服務** | 台灣最多高薪 AI Agent 職缺集中在 FinTech 和銀行（數位轉型） |
| **半導體** | 台積電、聯發科等 |
| **醫療** | 醫療 AI 應用 |
| **電商 / MarTech** | 聊天機器人、推薦系統 |
| **資安** | 趨勢科技等 |

> McKinsey 指出最稀缺的人才組合為：**法律 + AI 合規**、**醫療 + 具身 AI**、**金融 + AI 風險**、**HR + AI 轉型管理**

## 7. 經驗要求

| 等級 | 年資 | 備註 |
|---|---|---|
| 初階 | 1-2 年 | 純 LLM/Agent 職位較少開放初階 |
| **中階（最常見）** | **3-5 年** | 多數 AI Agent / LLM 職缺的最低門檻 |
| 資深 / Tech Lead | 5-8+ 年 | |
| 主管 / AI Lead | 8+ 年 | 需具備團隊管理經驗 |

> ⚠️ 多數 AI Agent 和 LLM 相關職位要求 **至少 3 年** 實務經驗，且強調 **生產環境經驗**（非僅原型開發）。

## 8. 2026 年新興趨勢

1. **AI Agent 編排（Orchestration）** — 多 Agent 協作系統設計
2. **MCP（Model Context Protocol）** — Anthropic 推出的 AI 工具整合標準，2026 年快速普及
3. **Agentic RAG** — 結合自主 Agent 與 RAG
4. **AI 評估 & 安全** — Red teaming、偏差偵測、輸出品質框架
5. **LLMOps** — 生產環境監控、prompt 版本管理、成本追蹤
6. **多模態 AI** — 視覺 + 語言模型整合
7. **Edge AI** — 在邊緣裝置部署模型（與台灣硬體生態系相關）
8. **AI 治理 & 倫理** — 負責任 AI 實踐，金融監管單位越來越重視
